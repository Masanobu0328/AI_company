# リサーチ

## 概要
市場調査・競合分析・業界トレンド・見込み客のWebサイト分析を保存するフォルダ。

## サブフォルダ構成
```
research/
├── market/         # 市場・業界調査
├── competitors/    # 競合分析
└── prospects/      # 見込み客サイト分析
```

## 対象コンテンツ
- 業界トレンドレポート
- 競合他社の分析
- ターゲット市場の調査
- 見込み客のWebサイト評価
- 技術トレンド

## 担当エージェント
- **営業エージェント** — 見込み客分析・競合調査
- **マーケティングエージェント** — 市場調査・トレンドリサーチ
