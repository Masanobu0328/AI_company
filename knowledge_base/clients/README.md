# クライアント

## 概要
見込み客・既存クライアントの情報・商談履歴を一元管理するフォルダ。

## フォルダ構成
```
clients/
├── README.md              # このファイル
└── [企業名]/              # クライアント別フォルダ
    ├── [企業名]_overview.md   # 基本情報・サマリー
    ├── outreach.md            # アウトリーチ・連絡履歴
    ├── discovery.md           # ディスカバリーメモ
    └── brief.md               # プロジェクトブリーフ（成約後）
```

## クライアントステータス
| ステータス | 説明 |
|---|---|
| prospect | 見込み客（未接触） |
| contacted | アウトリーチ済み |
| in_discussion | 商談中 |
| contracted | 成約・プロジェクト進行中 |
| completed | 納品完了 |
| archived | クローズ |

## 担当エージェント
- **営業エージェント** — クライアントフォルダの作成・更新
- **PMエージェント** — 成約後のプロジェクト情報追加
