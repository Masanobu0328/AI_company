## 概要
全エージェント（グラント/グレン/森岡/ウォズニアック）に進捗確認メッセージを送り、回答を集約してCEOに報告する。
各エージェントのDiscordチャンネルに `[MSG:チャンネル名: 内容]` 形式で問い合わせを送り、返答内容をまとめて一覧形式で報告する。

## 入力
- CEOからの進捗確認指示
- 現在並走中のプロジェクト・タスクの概要（memory.mdまたはlogsから確認）

## 実行手順
1. **問い合わせ先の特定**：現在稼働中のエージェントを確認する
2. **進捗確認メッセージを送信**：各エージェントへ `[MSG:チャンネル名]` 形式で問い合わせる
   - `[MSG:sales: グラント、現在の営業リスト作成・アウトリーチの進捗を教えてください]`
   - `[MSG:pm: グレン、プロジェクト全体の進捗と課題を教えてください]`
   - `[MSG:marketing: 森岡さん、マーケティング・分析の進捗を教えてください]`
   - `[MSG:dev: ウォズニアック、価格設計・LP制作の進捗を教えてください]`
3. **回答の収集**：各エージェントからの返答を記録する
4. **集約レポートの作成**：エージェントごとに現状・次のアクション・ブロック事項を整理する
5. **CEOへの報告**：Discord用フォーマットで簡潔にまとめて報告する
6. **未回答の処理**：返答がない場合は前回の記憶（memory.md）をもとに暫定情報を記載し、その旨を明示する

## 出力
- 全エージェント進捗サマリー（Discord用Markdownフォーマット）
- 保存先：`logs/YYYY-MM-DD_菅義偉-秘書.md` に記録

## 出力フォーマット例（Discord用）
```
📋 **進捗報告 - YYYY-MM-DD**
━━━━━━━━━━━━━━━━
💼 グラント（営業）：営業リストv3完成（20社）。TOP5特定済み。アウトリーチ文章作成待ち
🗂️ グレン（PM）：3タスク並走中。セグメント優先度・価格設計・事業モデル検討が進行中
📣 森岡（マーケ）：Threads毎日3本投稿中。セグメント分析に着手
💻 ウォズ（開発）：価格試算・LPモックアップがいずれも未着手
━━━━━━━━━━━━━━━━
```
