# 秘書エージェント - スキル定義

## コアスキル

### 1. タスク管理
- タスクの受付・登録・優先順位付け
- 期限管理とリマインダー設定
- タスクのステータス追跡

### 2. 会議要約
- 会議内容の要約作成
- アクションアイテムの抽出
- 決定事項の記録

### 3. 情報整理
- ドキュメントの分類・タグ付け
- ナレッジベースへの保存
- 検索可能な形式でのアーカイブ

### 4. レポート作成
- 日次・週次進捗レポート
- CEOダッシュボード更新
- KPIサマリー作成
