## 概要
見込み客リストに含まれる各ビジネスについて、Webサイトの有無・状態・SNS活用状況を調査する。
「サイトなし」「サイト古い」「Googleマップのみ」「SNSのみ」などのステータスを記録し、アプローチの優先順位付けとアウトリーチ文章のカスタマイズに活用する。

## 入力
- 見込み客リスト（ビジネス名・業種・Instagram URL等）

## 実行手順
1. **URL確認**：ビジネス名でGoogle検索し、公式サイトが存在するか確認する
2. **サイト状態の評価**：サイトが存在する場合、以下を確認する
   - 最終更新日（3年以上前 → 「古い」判定）
   - モバイル対応の有無
   - 問い合わせフォームの有無
   - デザインの品質（テキストのみ・写真少ない等）
3. **Googleマップ確認**：Googleマップ掲載の有無・レビュー数・写真数を記録する
4. **SNS状況確認**：Instagram・Facebook等のフォロワー数・投稿頻度・プロフィールリンクの有無を確認する
5. **ステータス分類**：以下のいずれかに分類する
   - `A: サイトなし + SNSあり`（最優先）
   - `B: サイト古い + SNSあり`（優先）
   - `C: Googleマップのみ`（優先）
   - `D: サイトあり（最新）`（後回し）
6. **結果を営業リストに反映**：ステータス列を更新する

## 出力
- 各ビジネスのWebステータスが記録された更新済み営業リスト
- 保存先：`projects/goldcoast/sales/prospect-list.md`（更新）
- 個別詳細シート：`projects/goldcoast/sales/clients/[ビジネス名].md`
