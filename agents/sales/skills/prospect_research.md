## 概要
ゴールドコースト在住の日本人オーナーが経営する小中規模ビジネス（飲食・カフェ・ネイル・フィットネス・整体・小売等）をWeb検索とInstagram検索で発掘する。
サイトなし・古いサイト・SNSのみで活動している事業者を優先的にリストアップし、営業ターゲットリストとしてまとめる。

## 入力
- 検索対象の業種（例：飲食・ネイル・フィットネス・整体・カフェ等）
- 対象地域（ゴールドコースト）
- 優先条件（サイトなし / サイト古い / Instagram保有 / 日本人オーナー）

## 実行手順
1. **Web検索でビジネスを発掘**：以下のキーワードで検索する
   - `"Gold Coast" + 業種 + "Japanese"`
   - `"Gold Coast" Japanese restaurant / nail salon / personal trainer`
   - Google マップで「Gold Coast 日本食 / ネイル / 整体」等を検索
2. **Instagram検索**：ハッシュタグ（#goldcoast #ゴールドコースト #goldcoastjapanese 等）で小規模ビジネスアカウントを探す
3. **Webサイト状況の確認**：各ビジネスについてサイト有無・最終更新日・Googleマップのみの掲載有無を確認する
4. **リスト化**：以下の項目を記入する
   - ビジネス名・オーナー名（判明分）
   - 業種カテゴリ
   - Instagram URL / Webサイト URL
   - Webサイト状況（なし / 古い / あり）
   - 優先度スコア（高/中/低）
5. **高優先度TOP5〜10の特定**：サイトなしかつInstagram保有の日本人オーナービジネスを最優先とする

## 出力
- 営業リスト（Markdown形式、20〜30件目安）
- 保存先：`projects/goldcoast/sales/prospect-list.md`
- 高優先度TOP5リストとその選定理由
