## 概要
ゴールドコーストにおけるWeb制作・デジタル支援の競合状況を調査し、自社の差別化ポイントを明確化する。
「日本語対応・日系ビジネス特化・バイリンガルサイト」という強みを軸に、競合が手を出していないポジションを特定する。
分析結果はSNS戦略・アウトリーチ文章・LP設計の土台として活用する。

## 入力
- ゴールドコースト地域のWeb制作会社・フリーランスの情報（Web検索で収集）
- 自社の提供サービス概要（HP作成支援・日本語対応・日系ターゲット特化）
- ターゲット顧客のペインポイント（サイトなし・古い・英語のみで対応困難等）

## 実行手順
1. **競合調査**：Web検索で「Gold Coast web design」「Gold Coast website builder」等を検索し、主要競合5〜10社を特定する
2. **競合マッピング**：各競合を「価格帯」×「日本語対応可否」×「ターゲット業種」で分類する
3. **自社強みの整理**：日本人オーナーへの日本語コミュニケーション・バイリンガルサイト対応・業種特化テンプレートなど差別化要素を列挙する
4. **ポジショニングマップ作成**：競合との位置関係を2軸グラフ（価格帯×専門性）で可視化する
5. **UVP（独自価値提案）を定義**：1〜2文で自社の立ち位置を言語化する
6. **活用指針の作成**：SNS投稿・アウトリーチ文章・LPに使えるメッセージラインを3パターン作成する

## 出力
- ポジショニング分析レポート（Markdown形式）
- 保存先：`projects/goldcoast/marketing/positioning.md`
- UVP定義文（SNS・LP・アウトリーチに流用可能な形）
