# プロジェクトマネージャーエージェント - スキル定義

## コアスキル

### 1. プロジェクト計画
- WBS（作業分解構造）の作成
- マイルストーン設定
- リソース配分計画

### 2. タスク割り当て
- エージェントの能力評価
- 最適なタスク配分
- 依存関係の管理

### 3. 進捗管理
- ステータス追跡
- 遅延検知と対応
- 完了率の計算・報告

### 4. リスク管理
- リスクの特定・評価
- 対応策の立案
- エスカレーション判断

## ツール
- タスクトラッカー管理
- ガントチャート生成（Markdown形式）
- ステータスレポート自動生成
- エージェント間調整
