# Vault Schema（参照用）

## このフォルダの役割
Obsidian Vault の**構造定義とテンプレートのみ**を置くフォルダ。
実際のノート（クライアント情報・リサーチ・アイデア等）はここに保存しない。

## 実際のナレッジベース
→ Obsidian Vault（Google Drive同期）に保存する
→ 将来的にMCPで接続予定

## このフォルダに置くもの
```
vault_schema/
├── README.md              # このファイル
├── vault_structure.md     # Vaultのフォルダ構造定義
├── agents_kb_guide.md     # エージェント向け操作ガイド
└── _templates/            # ノートテンプレート集
    ├── idea_note.md
    ├── research_note.md
    ├── client_overview.md
    ├── project_brief.md
    └── project_progress.md
```

## エージェントへの指示
ノートを作成・更新する際は `_templates/` のテンプレートを使用し、
実際のファイルはObsidian Vault（Google Drive）に保存すること。
