# ゴールドコースト ホームページ作成支援事業

作成日: 2026-03-13
ステータス: 進行中

---

## 事業概要

ゴールドコースト内の小中規模ビジネス・個人事業主を対象に、ホームページ作成支援を行うBtoB/BtoC事業。

## ターゲット

- 小中規模ビジネス（飲食・美容・不動産・フィットネス等）
- 個人事業主・フリーランサー
- Instagram等で活動しているトレーダー・個人アカウント
- Webサイトが古い・存在しない事業者を優先

## 提供サービス

- ランディングページ（LP）作成
- ホームページ設計・制作
- SNSマーケティング連動設計

## 各エージェントの役割

| エージェント | 担当 |
|---|---|
| グラント・カードン（営業） | 営業リスト作成・情報収集・アプローチ文章 |
| スティーブ・ウォズニアック（開発） | LP設計・コーディング・デザイン |
| 森岡毅（マーケティング） | 市場分析・SNS戦略・LP構造提案 |
| グレン・スターンズ（PM） | 全体進行・タスク管理・引き継ぎ調整 |
| 菅義偉（秘書） | CEO対応・情報整理・スケジュール管理 |

## フォルダ構成

```
projects/goldcoast/
  brief.md              ← このファイル
  sales/
    prospect-list.md    ← 営業リスト（グラントが作成）
    clients/            ← クライアント別情報シート
    outreach/           ← アプローチ文章テンプレート
  dev/
    _templates/         ← 業種別LPテンプレート
    [クライアント名]/   ← 個別LP（index.html, style.css等）
  marketing/
    positioning.md      ← ポジショニング分析
    sns-strategy.md     ← SNS戦略書
    lp-structure.md     ← LP構造提案
    competitor-research.md ← 競合調査
```
